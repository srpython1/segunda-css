1¿Qué es CSS? CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML.

2¿Cuál es la función principal de CSS? Su función principal es separar el contenido del documento de su diseño, permitiendo modificar la apariencia visual sin cambiar el HTML.

3¿Cómo se puede incluir CSS en un documento HTML? Se puede incluir CSS de tres maneras: inline (dentro de una etiqueta HTML), internal (dentro de un style en la cabecera) y external (en un archivo .css vinculado con link).

4¿Qué son los selectores en CSS? Los selectores son patrones utilizados para seleccionar los elementos HTML que se desea estilizar.

5¿Qué son las propiedades y valores en CSS? Las propiedades son características que se pueden aplicar a los elementos (como color o font-size), y los valores son las configuraciones específicas que se les asignan (como red o 16px).

6¿Cómo se aplica el modelo de caja en CSS? El modelo de caja describe cómo se comportan los elementos en términos de tamaño, incluyendo márgenes, bordes, relleno y el contenido en sí.

7¿Qué es el "cascading" en CSS? El "cascading" se refiere a la forma en que los estilos se aplican y se resuelven cuando hay múltiples reglas aplicables, priorizando según la especificidad y el orden.

8¿Qué son las media queries? Las media queries son condiciones que permiten aplicar estilos específicos según características del dispositivo, como el ancho de la pantalla.

9¿Cuáles son algunos estilos comunes que se pueden aplicar? Estilos comunes incluyen colores de fondo, fuentes, márgenes, alineación de texto, y bordes.

10¿Cómo se puede usar CSS para crear un diseño responsivo? Se utilizan media queries para ajustar el diseño a diferentes tamaños de pantalla, junto con unidades relativas como porcentajes y vw/vh para elementos flexibles.
